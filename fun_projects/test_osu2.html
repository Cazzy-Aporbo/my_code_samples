About the Course

Welcome! This course helps you think like a data scientist: follow claims back to evidence, choose appropriate methods, and communicate limitations honestly. Our software expectations are modest (by design). We’ll learn the essentials with **JASP**, **Desmos**, and **StatKey**; students who want depth can use **R** or **Python**—both optional and beginner-friendly.

Format
Short demos → guided practice → discussion
Assessment
Low-stakes practice + clear write-ups
Goal
Explain what the data truly supports

Course Topics (with the “why”)

Evidence & Arguments

  • Claims vs. evidence; operationalizing questions; data provenance
  • Observational vs. experimental designs; randomization & control
  • Reproducibility, transparency, and open science

Why: avoid asserting more than data can justify; make work checkable.

Distributions & Variation

  • Center, spread, shape; outliers & robust summaries
  • Visualization choices and how charts can mislead

Why: “average” often hides important structure.

Inference Basics

  • Confidence intervals and p-values (what they mean & what they don’t)
  • t-tests, ANOVA, correlation—used responsibly

Why: quantify uncertainty instead of guessing.

Resampling Logic

  • Randomization tests; bootstrap intervals (StatKey)
  • Simulation to validate intuition

Why: distribution-free insight builds trust.

Modeling & Assumptions

  • Simple regression; residuals and diagnostics
  • Confounding, Simpson’s paradox, leakage

Why: models explain, assumptions constrain.

Ethics & Communication

  • Consent, privacy, bias & fairness
  • Plain-language reporting; visual honesty

Why: clarity and integrity are part of the grade.

Key Definitions (fast reference)

Confidence Interval (CI)
Range of plausible values for a population parameter based on a sample.
p-value
Probability of seeing data as or more extreme if the null hypothesis were true.
Effect Size
Magnitude of a relationship/difference (e.g., Cohen’s d, r). Complements p-values.
Confounding
A hidden factor that influences both X and Y, distorting the relationship.
Randomization Test
Significance by shuffling labels to simulate the null world.
Bootstrap
Resample with replacement to estimate variability (e.g., CI).
Overfitting
Model memorizes noise; looks great on training data, fails on new data.
Reproducibility
Independent analysts can obtain the same results given your steps and data.

GUI & Web (no coding)

JASP — point-and-click statistics
Windows
  1. Download: jasp-stats.org/download
  2. Run installer → allow defaults
  3. Launch JASP
macOS
  1. Download the macOS .dmg
  2. Open and drag JASP into Applications
  3. Right-click → Open if Gatekeeper warns

  • Good for: t-tests, ANOVA, correlation, simple regression, APA tables

Smoke test: File → Open → load bds211_sleep_study.csv → run a t-test.

Desmos — quick graphs
  • Web: desmos.com/calculator
  • Docs: Help Center
  • Good for: distributions, lines, sanity checks
StatKey — resampling (browser)
  • Web: lock5stat.com/StatKey
  • Good for: bootstrap CIs, randomization tests
  • Tip: start with built-in datasets; then upload CSV

Recommended Workflows

JASP (No-code)

  1. Import CSV → inspect columns & missingness
  2. Describe → Descriptives (plots + summary)
  3. Run test (t, ANOVA, corr) → check assumptions
  4. Export APA table → write a 3-sentence interpretation

StatKey (Concept demo)

  1. Select procedure → choose example data
  2. Bootstrap or randomization → visualize sampling
  3. Capture CI/p-value → compare to JASP/Python

R (Reproducible)

  1. readr::read_csv() → dplyr clean
  2. ggplot2 plot → check assumptions
  3. t.test(), aov() or lm()
  4. Report effect size + CI in plain English

Python (Notebook)

  1. pandas.read_csv → .info(), .describe()
  2. matplotlib/seaborn plots
  3. scipy.stats or statsmodels for tests/models
  4. Narrate results + attach figure

FAQ

I’m new to stats and coding. Will I be okay?

Yes. You can complete the course using JASP + StatKey + Desmos. R/Python are optional for students who want extra control or plan to pursue data-heavy fields.

How do I install these on my home computer?

Follow the steps under Tools & Installs — Mac & Windows. If your OS blocks an app (Gatekeeper on macOS or SmartScreen on Windows), right-click the app and choose Open to allow it.

What if my laptop is underpowered?

Use web tools (Desmos, StatKey) and campus machines for heavier tasks if needed. JASP and R are lightweight on most laptops.

What file formats do we use?

CSV for datasets; PDF/DOCX for short write-ups; optional: .jasp (JASP), .R (R), .ipynb (Jupyter).

How are assignments graded?

On clarity of reasoning and correct interpretation—including limitations and assumptions—not just calculations.

Accessibility & accommodations?

Contact Disability Access Services (DAS) and the instructor early so we can plan appropriate adjustments.
